# OptibisApp

OptibisApp adalah aplikasi Android Native berbasis Kotlin yang dikembangkan sebagai sistem *Internal Business Management* (Manajemen Bisnis Internal). Aplikasi ini dirancang untuk memfasilitasi operasional tim Customer Service (CS) mulai dari validasi pesanan masuk, pengelolaan database klien, hingga pembuatan laporan kerja harian. Proyek ini dibuat untuk memenuhi tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman Mobile 1 di Universitas Teknologi Bandung.
